# Discord Command Reference - Copy & Paste Ready

## MESSAGE 1: ADMIN COMMANDS
```
📋 **ADMIN COMMANDS** (Requires Manage Roles)

**Boss Roles:**
• `/set-boss-role role:@Role` - Set role for all bosses
• `/multi-roles enable` - Enable tier-specific roles
• `/multi-roles set-boss tier:Tier1 role:@Role` - Set tier role
• `/view-settings` - View current configuration

**POG System:**
• `/set-pog-channel channel:#pog` - Set POG alert channel
• Detects drops with hearts > 99 automatically

**Test:** Trigger boss spawn or high-value drop
```

## MESSAGE 2: USER COMMANDS
```
👤 **USER COMMANDS**

**Notifications:**
• `/notifications view` - View your settings
• `/notifications set type:stamina enabled:true` - Toggle notifications
• `/dm enable type:stamina` - Get DMs instead of channel pings

**Types:** expedition, stamina, raid, raidSpawnReminder, drop

**Other:**
• `/suggestion` - Send feedback to bot owner
• `/help` - Interactive help menu

**Test:** Set notification, then trigger that event
```

## MESSAGE 3: CARD SEARCH & LEADERBOARD
```
🔍 **CARD SEARCH**
• `@bot f naruto` - Search 1000+ cards
• `@bot find fire duelist` - Search with filters
• Type number (1,2,3) to select from multiple results

📊 **LEADERBOARD**
• `rlb` - View top 10 droppers
• Click "Rare Drops" for Exotic/Legendary stats

**Test:** Search a card, check leaderboard after drops
```

## MESSAGE 4: WISHLIST SYSTEM
```
💝 **WISHLIST** (Max 10 cards)

• `wa Naruto` - Add card
• `wa Ichigo, Luffy, Goku` - Add multiple
• `wl` - View your wishlist
• `wr Naruto` - Remove card

**Auto-Notify:** Get pinged when wishlisted card spawns in raid

**Test:** Add cards, wait for raid with those cards
```

## MESSAGE 9: TROUBLESHOOTING
```
🐛 **TROUBLESHOOTING**

**Bot not responding:**
• Check bot is online
• Verify "Send Messages" permission
• Try slash command instead of mention

**Role not pinged:**
• Run `/view-settings` to verify role
• Check role is mentionable
• Verify bot has mention permission

**Reminders not working:**
• Check `/notifications view`
• Verify notification type enabled
• Check DM permissions if DM enabled

**POG not working:**
• Verify channel set with `/set-pog-channel`
• Check bot has send permission in channel
• Ensure hearts > 99

**Cache issues:**
• Run `rcache` to refresh manually
• Check `rcache stats` for info
```
